What it is: The line that wraps around the padding and content. Role: This is the physical boundary of the element. You can style it with width, color, and different styles

The transparent space immediately surrounding the content. Role: Padding is inside the border. It acts as "breathing room," pushing the edge of the border away from the content. If you give a box a background color, that color will extend into the padding area.

Content

The innermost box where your actual "payload" lives—this is where your text, images, or child elements appear. This defines the base size of your element before any spacing is added.
